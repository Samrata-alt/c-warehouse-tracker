Warehouse Shelf Tracker (C Project)

A simple and interactive warehouse shelf management system built using the C programming language. The program uses a fixed-size array to simulate 10 storage slots and allows users to add, remove, view, and manage shelf items through a clean menu-driven interface.

Features

1.Add item to a specific shelf slot

2.Remove item from a slot

3.View complete shelf status

4.Find the first empty slot

5.Count total empty slots

6.Clear the entire shelf

7.Full and empty shelf warnings

8.User-friendly menu system
